How it works

Find, verify, organize.

  1. 1

    Start from Home or Explore

    Check your overview, tap Plan a Trip, or search a destination directly from Explore.

  2. 2

    Open destination context

    Use destination cards, featured places, or categories to move naturally into route planning.

  3. 3

    Check Terminals when route certainty matters

    Review verified terminal details, source cues, confidence cues, and report corrections when needed.

  4. 4

    Build the trip in Plans

    Create, save, share, revisit, and organize plans with reminders and history.

  5. 5

    Return through Profile

    Manage account tools, Friends, Saved Places, Settings, and replayable Guide Mode access.

Guide Mode

A simple five-tab onboarding guide.

Guide Mode teaches Home, Explore, Terminals, Plans, and Profile through Welcome, Home, Explore, Terminals, Plans, Profile, and Finish steps.

  • Uses Skip, Back, Next, and Finish controls
  • Does not require real route searches, plans, friends, or demo data
  • Can be replayed from Settings or Profile when enabled
  • Keeps onboarding focused on the new five-tab structure
Seven steps

Welcome → Home → Explore → Terminals → Plans → Profile → Finish

A lightweight walkthrough that explains the app shell first, then lets commuters continue into the real app without needing sample activity.

Mission

TripLine PH is dedicated to delivering trustworthy, low-cost, accessible, and affordable transportation-related data to support users traveling between locations in the Philippines. The service provides accurate public transportation information, including transport routes, transport options, and destination information. This allows users to identify available transport routes, evaluate travel costs, and follow structured transport route instructions to reach their destinations.

Vision

To establish TripLine PH as the Philippines’ most dependable and easily navigable travel search system, providing clear and understandable transportation information to commuters and low-budget tourists.

    Expo-ready reliability

    Built to stay useful when the network gets noisy.

    HalaPH uses cache-first data loading, reduces repeated Firebase and Google activity during demos, keeps logs quieter, and uses safer weak-network fallbacks without promising perfect offline accuracy.

    • Uses cached and fallback data when network is weak
    • Reduces repeated cloud calls during demos
    • Keeps the app responsive when non-critical data fails to refresh
    • Uses quieter logs so demo behavior is easier to follow
    • Keeps usable fallback data when possible

    FAQ

    Common FAQs

    What is HalaPH?

    HalaPH is a Philippine commute planning app for routes, fares, trip plans, and collaboration.

    Is it free to use?

    Yes. HalaPH is intended to be free to download, with controlled sponsored content managed through the admin system.

    What transport modes does it support?

    HalaPH presents walking, jeepney, bus, train, and mixed route sequences where supported by the app data.

    Can I use it on Android?

    Yes. Use the Android download button or scan the APK QR code on this page.

    Can I install it on iPhone?

    Public iPhone installation is not available for this showcase. The iOS build is available as a live presenter-device demo.

    What is Guide Mode?

    It is a simple five-tab onboarding guide for Home, Explore, Terminals, Plans, and Profile. It uses Welcome through Finish steps and can be replayed from Settings or Profile when enabled.

    Are fare estimates exact?

    No. Fare estimates may vary depending on route changes, distance, passenger type, and updated fare rules.

    Can I plan trips with friends?

    Yes. HalaPH supports shared plans with friends, participants, and starting points.
